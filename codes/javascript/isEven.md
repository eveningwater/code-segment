| isEven(是否是偶数) | math(数学) |

检查给定的数字是否是偶数。

- 使用模 (%) 运算符检查数字是奇数还是偶数。
- 如果数字是偶数，则返回 true，如果数字是奇数，则返回 false。

```js
const isEven = num => num % 2 === 0;
```

> 调用方式:

```js
isEven(3); // false
```
