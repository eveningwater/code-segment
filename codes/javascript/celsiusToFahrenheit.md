| celsiusToFahrenheit (转换数组) | math,beginner(数学，初学者) |

将摄氏度转换为华氏度。

- 遵循转换公式 F = 1.8 \* C + 32。

> 代码如下:

```js
const celsiusToFahrenheit = degree => 1.8 * degree + 32;
```

> 调用方式:

```js
celsiusToFahrenheit(33); // 91.4
```
