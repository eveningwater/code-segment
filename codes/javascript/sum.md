| sum(求和) | math(数学) |

计算两个或多个数字/数组的总和。

- 使用 Array.prototype.reduce() 将每个值添加到一个累加器，用值 0 初始化。

```js
const sum = (...arr) => [...arr].reduce((acc, val) => acc + val, 0);
```

> 调用方式:

```js
sum(1, 2, 3, 4); // 10
sum(...[1, 2, 3, 4]); // 10
```
