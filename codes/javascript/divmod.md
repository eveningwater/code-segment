| divmod(求两个数的商和模) | math(数学) |

返回由给定数字的商和余数组成的数组。

* 使用 `Math.floor()` 获得除法 `x / y` 的商。
* 使用模运算符 (`%`) 获得除法 `x / y` 的余数。

> 代码如下:

```js
const divmod = (x,y) => [Math.floor(x / y),x % y];
```

> 调用方式:

```js
divmod(8, 3); // [2, 2]
divmod(3, 8); // [0, 3]
divmod(5, 5); // [1, 0]
```
