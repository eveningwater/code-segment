| milesToKm(英里转换成公里) | math(数学) |

将英里转换为公里。

* 按照换算公式`km = mi * 1.609344`。

```js
const milesToKm = miles => miles * 1.609344;
```

<div class="code-editor" data-url="codes/javascript/ts/miles-to-km.ts" data-language="typescript"></div>

> 调用方式:

```js
milesToKm(5); // ~8.04672
```
