| take(截取元素数组) | array(数组) |

创建一个从开头移除 n 个元素的数组。

- 使用 Array.prototype.slice() 创建数组的一个切片，其中 n 个元素从头开始。

```js
const take = (arr, n = 1) => arr.slice(0, n);
```

> 调用方式:

```js
take([1, 2, 3], 5); // [1, 2, 3]
take([1, 2, 3], 0); // []
```
