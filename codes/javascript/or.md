| or(逻辑或) | math,logic(数学，逻辑) |

检查至少一个参数是否为真。

* 对两个给定值使用逻辑或 (||) 运算符。

```js
const or = (a,b) => a || b;
```


<div class="code-editor" data-url="codes/javascript/ts/or.ts" data-language="typescript"></div>

> 调用方式:

```js
or(true, true); // true
or(true, false); // true
or(false, false); // false
```
