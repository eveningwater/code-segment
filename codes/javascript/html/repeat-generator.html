📖 函数说明

repeatGenerator 函数

const repeatGenerator = function* (val) { let v = val; while (true) { let newVal = yield v; if (newVal !== undefined) { v = newVal; } } };

repeatGenerator 是一个无限生成器函数，它会持续生成当前值。 通过 next() 获取当前值，通过 next(newValue) 更新内部值。

使用示例

const gen = repeatGenerator(10); // 获取当前值 console.log(gen.next()); // { value: 10, done: false } // 更新值并获取 console.log(gen.next(20)); // { value: 20, done: false } // 再次获取（值保持为20） console.log(gen.next()); // { value: 20, done: false }
