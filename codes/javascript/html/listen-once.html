📖 函数说明

listenOnce 函数（监听一次事件）

const listenOnce = (el, type, handler) => 
  el.addEventListener(type, handler, { once: true });

函数说明

listenOnce 函数用于将事件监听器添加到仅在第一次触发事件时运行回调的元素。 使用 EventTarget.addEventListener() 将事件监听器添加到元素。 使用 { once: true } 作为选项，只运行一次给定的回调。 触发后监听器会自动移除，无需手动清理。

使用方式

// 基本用法
listenOnce(
  document.getElementById('my-button'),
  'click',
  () => console.log('只会触发一次')
);

// 多种事件类型
listenOnce(element, 'mouseenter', handler);
listenOnce(element, 'focus', handler);
listenOnce(element, 'keydown', handler);

// 事件对象
listenOnce(element, 'click', (e) => {
  console.log('事件目标:', e.target);
  console.log('触发时间:', new Date());
});

应用场景

  • 🎯 首次交互：捕获用户的首次点击或输入
  • 📊 数据采集：收集用户首次操作数据
  • 🎓 新手引导：只在首次展示提示信息
  • 💾 数据保存：首次编辑时自动保存
  • 🔔 通知提醒：首次进入页面时显示通知
  • 🎬 动画触发：元素首次进入视口时播放动画
  • 📝 表单验证：首次失焦时验证输入
  • 🚀 懒加载：首次滚动时加载资源

优势特点

  • ✅ 自动清理：无需手动移除监听器
  • ✅ 内存优化：避免重复触发和内存泄漏
  • ✅ 代码简洁：比手动管理更简单
  • ✅ 浏览器原生：使用原生 API，性能优秀

与普通监听器对比

// 普通监听器（需要手动移除）
const handler = () => {
  console.log('触发了');
  element.removeEventListener('click', handler);
};
element.addEventListener('click', handler);

// listenOnce（自动移除）
listenOnce(element, 'click', () => {
  console.log('触发了');
});

支持的事件类型

  • 鼠标事件：click, dblclick, mouseenter, mouseleave, contextmenu
  • 键盘事件：keydown, keyup, keypress
  • 焦点事件：focus, blur
  • 输入事件：input, change
  • 其他事件：scroll, resize, load 等
