📖 函数说明

levenshteinDistance 函数（编辑距离）

const levenshteinDistance = (s, t) => {
            if (!s.length) {
                return t.length;
            }
            if (!t.length) {
                return s.length;
            }
            const arr = [];
            for (let i = 0; i <= t.length; i++) {
                arr[i] = [i];
                for (let j = 1; j <= s.length; j++) {
      arr[i][j] = i === 0 ? j : Math.min(
        arr[i - 1][j] + 1,        // 删除
        arr[i][j - 1] + 1,        // 插入
        arr[i - 1][j - 1] + (s[j - 1] === t[i - 1] ? 0 : 1)  // 替换/匹配
      );
                }
            }
            return arr[t.length][s.length];
};

算法说明

Levenshtein Distance 是计算两个字符串之间差异的算法，也称为编辑距离。 它表示将一个字符串转换为另一个字符串所需的最少编辑操作（插入、删除或替换）次数。 使用动态规划算法，时间复杂度为 O(m×n)，空间复杂度为 O(m×n)。

使用方式

levenshteinDistance('kitten', 'sitting');  // 3
levenshteinDistance('duck', 'dark');        // 2
levenshteinDistance('hello', 'world');      // 4
levenshteinDistance('', 'abc');             // 3
levenshteinDistance('same', 'same');        // 0

应用场景

  • 🔍 搜索引擎：模糊搜索和拼写纠错
  • 📝 文本编辑器：自动纠错和建议功能
  • 🔬 生物信息学：DNA 序列比对和分析
  • 📊 数据清洗：重复数据检测和合并
  • 🎯 推荐系统：计算文本相似度
  • 🌍 机器翻译：译文质量评估
  • 📱 输入法：候选词排序和纠错
  • 🔒 密码强度：检测相似密码

算法步骤

  1. 初始化：创建 (m+1)×(n+1) 的二维数组
  2. 边界条件：第一行和第一列初始化为递增序列
  3. 状态转移：对于每个格子 (i,j)，计算三种操作的最小值：
    • 删除：dp[i-1][j] + 1
    • 插入：dp[i][j-1] + 1
    • 替换/匹配：dp[i-1][j-1] + cost（相同为0，不同为1）
  4. 返回结果：右下角格子的值即为编辑距离

相似度计算

相似度 = (1 - 编辑距离 / max(字符串1长度, 字符串2长度)) × 100%
这个值越接近 100%，说明两个字符串越相似。
