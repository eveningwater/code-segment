📖 函数说明

lowercaseKeys 函数（小写键名）

const lowercaseKeys = obj => Object.keys(obj).reduce((acc, key) => {
    acc[key.toLowerCase()] = obj[key];
    return acc;
}, {});

函数说明

lowercaseKeys 函数将对象的所有键名转换为小写。 使用 Object.keys() 获取所有键，然后通过 Array.prototype.reduce() 遍历每个键， 将键名转换为小写并创建新对象。原始对象的值保持不变。

使用方式

const myObj = { Name: 'Fred', Age: 42 };
lowercaseKeys(myObj); // { name: 'Fred', age: 42 }

const config = { API_KEY: 'abc123', USER_NAME: 'john' };
lowercaseKeys(config); // { api_key: 'abc123', user_name: 'john' }

应用场景

  • 🔄 API 数据标准化：统一不同来源的数据键名格式
  • 📊 数据库查询：将查询结果键名转换为标准格式
  • ⚙️ 配置文件处理：标准化配置项的键名
  • 🔍 键名查找：不区分大小写的键名查找
  • 📝 表单数据处理：统一表单字段名称格式
  • 🌐 国际化：处理不同语言环境的键名
  • 🎯 数据导入：处理外部数据的键名格式
  • 🔐 环境变量：标准化环境变量名称

特性说明

  • ✅ 不可变操作：返回新对象，不修改原对象
  • ✅ 保留值：只转换键名，值保持原样
  • ✅ 简洁高效：使用函数式编程方法
  • ✅ 通用性强：适用于任何 JavaScript 对象
