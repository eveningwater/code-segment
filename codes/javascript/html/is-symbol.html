📖 Symbol 说明

isSymbol 函数

const isSymbol = val => typeof val === 'symbol';

Symbol 特性

  • 唯一性：每个 Symbol 都是唯一的，即使描述相同
  • 不可变：Symbol 值创建后不能被修改
  • 隐藏性：Symbol 属性不会出现在 for...in 循环中
  • 全局注册：可以通过 Symbol.for() 创建全局共享的 Symbol
  • 内置 Symbols：JavaScript 提供多个 Well-known Symbols

常见用途

  • 创建对象的唯一属性键，避免命名冲突
  • 定义对象的元编程行为（如 Symbol.iterator）
  • 实现私有属性（虽然不是真正的私有）
  • 创建常量枚举值
