📖 使用说明

关于 Session Storage

  • 会话级别：数据仅在当前页面会话期间有效，关闭标签页后数据会被清除
  • 存储容量：通常为 5-10 MB（因浏览器而异）
  • 作用范围：同一标签页内的所有页面共享数据
  • 数据类型：只能存储字符串，其他类型需要序列化
  • 安全性：数据存储在客户端，不应存储敏感信息

功能特性

  • ✅ 实时检测 Session Storage 可用性
  • 📝 支持单个和批量数据操作
  • 🔍 JSON 数据可视化和搜索
  • 📊 存储容量统计和监控
  • 📤 数据导入导出功能
  • 🎨 友好的用户界面和动画效果
