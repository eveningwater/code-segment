📖 函数说明

isString 函数

const isString = val => typeof val === 'string';

功能说明

  • 使用 typeof 操作符检测值的类型
  • 返回 true 表示值为字符串类型
  • 返回 false 表示值不是字符串类型
  • 支持检测任意 JavaScript 值

功能特性
