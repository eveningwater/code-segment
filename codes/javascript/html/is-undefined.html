Undefined 出现场景

  • 未声明的变量：访问未声明的变量会抛出错误
  • 声明但未赋值：let x; // x 为 undefined
  • 对象不存在的属性：obj.nonExist // undefined
  • 函数无返回值：没有 return 或 return 后没有值
  • 函数参数未传递：调用时未提供的参数
  • void 运算符：void 0 返回 undefined

Undefined vs Null

// undefined: 变量未初始化或不存在
let x;
console.log(x); // undefined

// null: 明确表示"无值"
let y = null;
console.log(y); // null

// 比较
console.log(undefined == null);  // true (宽松相等)
console.log(undefined === null); // false (严格相等)
