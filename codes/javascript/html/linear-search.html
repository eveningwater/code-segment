📖 函数说明

linearSearch 函数（线性搜索）

const linearSearch = (arr, item) => {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] === item) {
      return +i;  // 返回索引
    }
  }
  return -1;  // 未找到
};

算法说明

线性搜索（Linear Search）是最简单直接的搜索算法。 它从数组的第一个元素开始，逐个检查每个元素是否等于目标值，直到找到目标或遍历完整个数组。 时间复杂度为 O(n)，空间复杂度为 O(1)。

使用方式

linearSearch([2, 9, 9], 9);     // 1
linearSearch([2, 9, 9], 7);     // -1
linearSearch([5, 2, 8, 1], 2);  // 1
linearSearch([1, 2, 3], 4);     // -1

应用场景

  • 🔍 小规模数据：数据量较小时的简单搜索
  • 📊 无序数组：数组未排序或无法排序时
  • 🎯 单次搜索：只需要搜索一次的场景
  • 💾 内存限制：空间复杂度要求严格时
  • 📝 链表搜索：链表等顺序访问的数据结构
  • 🔄 动态数据：频繁插入删除的数据集

算法特点

  • ✅ 简单直观：实现简单，易于理解
  • ✅ 通用性强：适用于任何数据结构
  • ✅ 空间效率：O(1) 空间复杂度
  • ⚠️ 时间复杂度：O(n) 最坏情况
  • ⚠️ 效率较低：大数据量时性能差

性能分析

  • 最佳情况：O(1) - 目标在第一个位置
  • 平均情况：O(n/2) - 目标在中间位置
  • 最坏情况：O(n) - 目标在最后或不存在
  • 空间复杂度：O(1) - 只需常数级额外空间

与其他算法对比

  • vs 二分搜索：二分搜索更快（O(log n)），但需要有序数组
  • vs 哈希搜索：哈希搜索更快（O(1)），但需要额外空间
  • vs 插值搜索：插值搜索在均匀分布时更快
