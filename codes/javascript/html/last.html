📖 函数说明

last 函数

const last = arr => arr && arr.length ? arr[arr.length - 1] : undefined;

函数说明

last 函数用于获取数组中的最后一个元素。它首先检查数组是否存在且具有长度属性， 然后使用 arr.length - 1 计算最后一个元素的索引并返回该元素。 如果数组为空或不存在，则返回 undefined。

使用方式

last([1, 2, 3]);        // 3
last(['a', 'b', 'c']);  // 'c'
last([]);               // undefined
last(null);             // undefined
last(undefined);        // undefined

应用场景

  • 📊 数据分析：获取时间序列的最新数据点
  • 📝 日志处理：获取最后一条日志记录
  • 🔍 历史记录：获取用户最近的操作
  • 📈 状态管理：获取状态变更的最新状态
  • 🎮 游戏开发：获取玩家的最后一次操作
  • 💬 消息系统：获取会话的最后一条消息
