📖 函数说明

join 函数

const join = (arr, separator = ',', end = separator) => 
    arr.reduce((acc, val, i) => 
        i == arr.length - 2 ? acc + val + end : 
        i === arr.length - 1 ? acc + val : 
        acc + val + separator, 
    '');

参数说明

  • arr - 要连接的数组
  • separator - 元素之间的分隔符（默认: ','）
  • end - 最后两个元素之间的分隔符（默认: 与 separator 相同）

使用示例

// 基本用法
join(['a', 'b', 'c']); 
// 结果: 'a,b,c'

// 自定义分隔符
join(['apple', 'banana', 'orange'], ', '); 
// 结果: 'apple, banana, orange'

// 自定义结尾分隔符
join(['pen', 'pineapple', 'apple', 'pen'], ', ', ' and '); 
// 结果: 'pen, pineapple, apple and pen'

// 自然语言格式
join(['红色', '绿色', '蓝色'], '、', '和');
// 结果: '红色、绿色和蓝色'
