📖 函数说明

isWeekday 函数

const isWeekday = (d = new Date()) => d.getDay() % 6 !== 0;

函数原理

  • getDay() 返回 0-6，其中 0 = 周日，6 = 周六
  • 工作日（周一到周五）的 getDay() 返回 1-5
  • getDay() % 6 !== 0 用于排除周日(0)和周六(6)
  • 返回 true 表示是工作日
  • 返回 false 表示是周末
