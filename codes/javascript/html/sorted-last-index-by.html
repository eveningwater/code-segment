sortedLastIndexBy 实战：按迭代器求最后插入下标

sortedLastIndexBy(arr, n, fn) 会把元素映射到 fn 轴上的数值来比较，并返回保持排序的最高插入下标 （相等时插到最后一个相等元素之后）。\n 本页支持对象数组/数字数组、可视化插槽、批量探测、模拟插入、导出与源码高亮。

配置

选择字段后，比较基于 fn(el) 与 fn(n)；恒等模式适用于数字数组。
注意：本函数会“松散推断”升序/降序（比较首尾的 fn 值）。若数据本身不单调，结果仅供参考，可勾选检测。
