📖 函数说明

isUpperCase 函数

const isUpperCase = str => str === str.toUpperCase();

功能说明

  • 检测字符串是否全部为大写字母
  • 通过比较字符串与其大写形式是否相等来判断
  • 返回 true 表示全部大写
  • 返回 false 表示包含小写或其他字符
