主文档区块

简介

本页演示在长页面中通过选择器平滑滚动到各区块。顶部导航与左侧按钮共用同一套 smoothScroll 封装，便于在真实项目中统一行为与可访问性处理。

特性

支持自定义 block / inline、可选失败提示、滚动完成后的通知与高亮。 嵌套滚动区域可独立演示「文档不滚、只滚容器」的效果。

API

smoothScroll(selector, options?)：第二个参数可传 { block, inline, onMissing }。若元素不存在可回调或通知。

FAQ

固定顶栏会挡住标题时，请调大 scroll-margin-top 或为区块设置 CSS scroll-margin-top（本页通过变量绑定）。
