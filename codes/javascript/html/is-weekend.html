📖 函数说明

isWeekend 函数

const isWeekend = (d = new Date()) => d.getDay() % 6 === 0;

函数原理

  • getDay() 返回 0-6，其中 0 = 周日，6 = 周六
  • 周末（周六和周日）的 getDay() 返回 0 或 6
  • getDay() % 6 === 0 用于检测周六(6%6=0)和周日(0%6=0)
  • 返回 true 表示是周末
  • 返回 false 表示是工作日
