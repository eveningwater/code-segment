📖 函数说明

leftSubstrGenerator 函数（左侧子串生成器）

const leftSubstrGenerator = function* (str) {
  if (typeof str !== 'string' || !str.length) {
                return;
            }
            for (let i in str) {
    yield str.slice(0, parseInt(i) + 1);
  }
};

函数说明

leftSubstrGenerator 是一个生成器函数，用于生成给定字符串的所有左子字符串。 如果字符串为空，使用 String.prototype.length 提前终止。 使用 for...in 循环和 String.prototype.slice() 从头开始产生给定字符串的每个子字符串。

使用方式

[...leftSubstrGenerator('hello')];
// [ 'h', 'he', 'hel', 'hell', 'hello' ]

// 使用 for...of 遍历
for (const substr of leftSubstrGenerator('abc')) {
  console.log(substr);
}
// 'a'
// 'ab'
// 'abc'

// 手动迭代
const gen = leftSubstrGenerator('test');
console.log(gen.next().value); // 't'
console.log(gen.next().value); // 'te'
console.log(gen.next().value); // 'tes'
console.log(gen.next().value); // 'test'

  • 🔍 前缀搜索：实现字符串前缀匹配和搜索
  • 📝 自动补全：输入提示和自动完成功能
  • 🎯 模式匹配：逐步匹配字符串模式
  • 📊 数据分析：分析字符串的渐进特性
  • 🎨 动画效果：文字逐字显示动画
  • 🔤 拼写检查：逐步验证单词拼写
  • 🌳 前缀树构建：Trie 数据结构的构建
  • 🎮 游戏逻辑：猜词游戏的提示系统

Generator 特性

  • ✅ 惰性求值：按需生成，不会一次性占用大量内存
  • ✅ 可迭代：实现了 Iterator 协议，可用于 for...of
  • ✅ 状态保持：每次 yield 后保持函数状态
  • ✅ 无限序列：可以生成无限长度的序列

时间复杂度

生成所有子串的时间复杂度为 O(n²)，其中 n 是字符串长度。 每次 slice 操作需要 O(i) 时间，总共需要 1 + 2 + 3 + ... + n = n(n+1)/2 次操作。
