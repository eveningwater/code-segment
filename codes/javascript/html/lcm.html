📖 函数说明

LCM 函数（最小公倍数）

const lcm = (...arr) => {
            const gcd = (x, y) => (!y ? x : gcd(y, x % y));
            const _lcm = (x, y) => (x * y) / gcd(x, y);
            return [...arr].reduce((a, b) => _lcm(a, b));
};

函数说明

lcm 函数用于计算两个或多个数字的最小公倍数（Least Common Multiple）。 它使用最大公约数（GCD）公式和 lcm(x, y) = x * y / gcd(x, y) 的公式来确定最小公倍数。 GCD 公式使用递归实现。

使用方式

lcm(12, 7);              // 84
lcm(4, 6);               // 12
lcm(...[1, 3, 4, 5]);    // 60
lcm(12, 15, 20);         // 60

应用场景

  • 📅 时间周期：计算多个事件同时发生的周期
  • 🎵 音乐节拍：计算不同节拍的同步点
  • ⚙️ 齿轮计算：计算齿轮的公共转动周期
  • 📦 包装问题：计算最小包装单位
  • 🔢 分数运算：计算分数的公共分母
  • 🚌 公交调度：计算多条线路的发车周期

数学原理

  • ✅ GCD（最大公约数）：使用欧几里得算法递归计算
  • ✅ LCM 公式：lcm(a, b) = (a × b) / gcd(a, b)
  • ✅ 多数计算：使用 reduce 逐对计算
  • ✅ 时间复杂度：O(n × log(min(a, b)))
