| drop(截取数组（从左到右依次根据起始索引值截取）) | array,beginner(数组,初学者) |

返回一个新数组，其中从左侧删除了 $n 个元素。

* 使用 `array_slice()` 从左侧删除 $n 个元素。
* 省略第二个参数 $n，则仅删除一个元素。

> 代码如下:

```php
function drop($items, $n = 1){
    return array_slice($items, $n);
}
```

> 使用方式:

```php
drop([1, 2, 3]); // [2,3]
drop([1, 2, 3], 2); // [3]
```
