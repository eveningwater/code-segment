| head(返回数组头部元素(即第一个元素)) | array,beginner(数组，初学者) |

返回列表的头部。

- 使用 reset() 返回数组中的第一个元素。

> 代码如下:

```php
function head($arr){
    return reset($arr);
}
```

> 使用方式:

```php
head([1, 2, 3]); // 1
```
