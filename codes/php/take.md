| take(字符串开头匹配子字符串) | array,beginner(数组,初学者) |

返回一个从开头删除了 $n 个元素的数组。

* 使用 array_slice() 从数组开头删除 $n 个项目。

> 代码如下:

```php
function take($items,$n = 1){
    return array_slice($items,0,$n);
}
```

> 使用方式:

```php
take([1, 2, 3], 5); // [1, 2, 3]
take([1, 2, 3, 4, 5], 2); // [1, 2]
```
