| 标题                  | 标签                          |
| --------------------- | ----------------------------- |
| listStashes(存储列表) | repository,stash(存储库,存储) |

显示所有存储的列表。

- 使用 `git stash list` 查看所有存储的列表。

> 代码如下:

```shell
# 语法
git stash list
```

> 使用示例:

```shell
git stash list
# stash@{0}：WIP分支-1：ee52eda 修复网络错误
```
