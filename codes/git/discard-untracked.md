| 标题                               | 标签         |
| ---------------------------------- | ------------ |
| discardUntracked(放弃未追踪的更改) | branch(分支) |

放弃对当前分支的所有未跟踪的更改。

- 使用 `git clean -f -d`放弃当前分支的所有未跟踪更改。

> 代码如下:

```shell
# 语法
git clean -f -d
```

> 使用示例:

```shell
git clean -f -d
# 放弃所有未跟踪的更改
```
