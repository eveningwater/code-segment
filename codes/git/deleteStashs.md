| deleteStashs(删除所有的暂存区) | stash,repository(暂存区，仓库) |

删除所有暂存区。

- 使用 `git stash clear` 删除所有暂存区。

> 代码如下:

```shell
# 语法
git stash clear
```

> 使用示例:

```shell
git stash clear
# 删除所有的暂存区
```
