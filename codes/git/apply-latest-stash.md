| applyLatestStash(应用最近的存储) | repository,stash(仓库,存储) |

应用最新的存储。

- 使用 `git stash apply` 应用最新的存储。

> 代码如下:

```shell
git stash apply <stash>
```

> 使用示例:

```shell
git stash apply # apply the latest stash
# 在终端运行该命令，将应用最新的存储到当前分支
```
