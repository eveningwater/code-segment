| editConfig(编辑配置) | configuration(配置) |

在 git 文本编辑器中打开 git 配置文件。

- 使用 `git config --global -e` 在 git 文本编辑器中打开 git 配置文件。

> 代码如下:

```shell
# 语法
git config --global -e
```

> 使用示例:

```shell
git config --global -e
# 在默认的 git 文本编辑器中打开 git 配置文件
```
